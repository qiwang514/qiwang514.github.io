# Java 后端学习编程路线（循序渐进版）

## 阶段一：Java 基础（1-2 个月）
- **核心语法**：变量、数据类型、运算符、流程控制、数组、字符串
- **面向对象**：类与对象、封装、继承、多态、抽象类、接口、注解
- **集合框架**：List/Set/Map 核心 API、HashMap 底层原理、迭代器
- **异常处理**：try-catch-finally、自定义异常
- **IO 流**：文件读写、字节流/字符流、NIO 基础
- **多线程基础**：Thread、Runnable、线程状态、同步锁、volatile

## 阶段二：Java 进阶（1-2 个月）
- **JVM 基础**：内存模型、类加载机制、GC 原理、调优入门
- **并发编程**：JUC 包（Lock、线程池、CountDownLatch、Atomic）、线程安全
- **新特性**：Java 8+（Lambda、Stream、Optional、函数式接口）
- **网络编程**：Socket、HTTP 基础、IO 模型

## 阶段三：Web 开发（2 个月）
- **前端基础**：HTML/CSS/JS、Vue/React 入门
- **Servlet 与 Tomcat**：请求处理、会话管理
- **Spring 全家桶**：
    - Spring Core：IOC、AOP、Bean 生命周期
    - Spring Boot：自动配置、Starter、日志、配置文件
    - Spring MVC：请求映射、参数解析、RESTful 接口
- **数据库**：MySQL 基础、SQL 优化、事务、索引
- **ORM 框架**：MyBatis/MyBatis-Plus 核心用法

## 阶段四：微服务与高并发（2-3 个月）
- **微服务基础**：服务拆分、Nacos 注册中心、Gateway 网关
- **服务调用**：OpenFeign、负载均衡
- **分布式技术**：Redis（缓存、分布式锁）、RabbitMQ（消息队列）
- **高并发**：限流、降级、熔断、缓存雪崩/击穿/穿透
- **部署运维**：Docker、Jenkins、Linux 常用命令

## 阶段五：项目实战（1-2 个月）
- 选择一个完整项目（如博客系统、电商后台、AI 代码生成工具）
- 从需求分析 → 设计 → 编码 → 测试 → 部署全流程实践
- 沉淀项目经验，整理成简历
